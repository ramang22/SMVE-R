4 Testovanie štatistických hypotéz

Technicky, testovanie štat.hypotéz a konštrukcia intervalov spoľahlivosti používa rovnaký postup, rozdiel je len v závere. Pri testovaní nás zaujíma odpoveď na otázku: dá sa na základe dostupných pozorovaní zamietnuť (nulová) hypotéza?

4.1 Jedno-výberové testy o parametroch rozdelenia

4.1.1 Podiel

Príklad.

Vráťme sa k prieskumu obľúbenosti značky ABC medzi kupujúcimi, kde sa v ankete 42 ľudí zo 100 vyjadrilo, že má rado značku ABC. Môžeme tvrdiť, že práve polovica všetkých spotrebiteľov má rada ABC?
Riešenie:
Najprv treba formulovať nulovú hypotézu H0: Parameter obľúbenosti výrobku pre základný súbor je \(p_0=50\%\). Alternatívna H1: \(p_0\neq 0.5\) v prípade obojstranného testu.
Ak platí H0, testovacia štatistika \(TS=\frac{p-p_0}{\sigma}\) má asymptoticky N(0,1) rozdelenie.

p0 <- 0.5  # parameter alt.rozdelenia za predpokladu platnosti H0
n <- 100  # sample size
alpha <- 0.05  # hladina významnosti
TS <- (42/100-p0) / sqrt(p0*(1-p0)/n)  # testovacia štatistika
c(test_statistika = TS, 
  krit_hodnota = qnorm(c(alpha/2, 1-alpha/2)),  # kvantil norm.rozd.
  p_hodnota = 2*pnorm(TS)
  )
## test_statistika   krit_hodnota1   krit_hodnota2       p_hodnota 
##      -1.6000000      -1.9599640       1.9599640       0.1095986

Testovacia štatistika padla medzi kritické hodnoty, preto na 5% hladine významnosti nemôžeme zamietnuť hypotézu o tom, že polovica ľudí má rada značku ABC. To je vidieť aj z porovnania p-hodnoty s hladinou významnosti. P-hodnota sa vypočíta ako plocha pod hustotou rozdelenia testovacej štatistiky (tu normálneho) “za” bodovým odhadom testovacej štatistiky (v zmysle “smerom ku chvostu” rozdelenia). V prípade obojstranného testu je p-hodnota dvojnásobok tej plochy.

x <- seq(-3, 3, length.out = 100)
dens <- dnorm(x)
ind <- list(l = which(x<=-abs(TS)), r = which(x>=abs(TS)))
plot(x, dens, type = "l", ylim = c(-0.05, 0.4), main = "Znázornenie p-hodnoty")
polygon(x = c(x[1], x[ind$l], x[tail(ind$l,1)]), y = c(0, dens[ind$l], 0), 
        col="lightblue", border = NA)
polygon(x = c(x[ind$r[1]], x[ind$r], tail(x,1)), y = c(0, dens[ind$r], 0), 
        col="lightblue", border = NA)
arrows(x0 = qnorm(alpha/2), y0=-0.025, x1 = qnorm(1 - alpha/2), y1 = -0.025, 
       angle = 90, code = 3, length = 0.15)
text(x = 0, y = -0.025, labels = "Oblasť nezamietnutia H0", pos = 3)

Použitie alternatívnych testov:

binom.test(42, 100, p=0.5)

4.1.2 Stredná hodnota

Príklad

Výrobca modelu auta deklaruje dojazd 25mpg (míľ na galón paliva). Z prieskumnej vzorky 10 vlastníkov áut daného modelu bol zistený priemerný dojazd 23mpg a štandardná odchýlka 2.5mpg. Dá sa výrobcovi veriť?
Riešenie:

xmean <- 23; xsd <- 2.5
( TS <- (xmean-25)/(xsd/sqrt(10)) )
## [1] -2.529822
( pt(TS, df=10-1) )  # p-hodnota < 0.05
## [1] 0.01612239

H0 o rovnosti skutočného a deklarovaného dojazdu musíme zamietnuť.
Situácia tentokrát pomocou ggplot:

library(ggplot2)
data.frame(x = seq(-3, 3, length.out = 100)) %>%  # mriežka x-ových hodnôt
  dplyr::mutate(dens = dt(x, df=10-1)) %>%   # pridá y-ove
  ggplot(mapping = aes(x = x, y = dens)) +  
  geom_line() + 
  geom_ribbon(mapping = aes(x = ifelse(x < TS, x, NA), ymin = 0, ymax = dens), 
              fill = "lightblue") +  # pridá plochu = p-value
  coord_cartesian(ylim=c(-0.05,0.4)) +  # trochu miesta pod grafom
  geom_segment(x = qt(0.05, df=10-1), y = -0.025, xend = max(x), yend = -0.025, 
               arrow = arrow(angle=90, length = unit(0.15, "inches"), ends = "first"),
               size = 0.1) +  # úsečka s jedným ukončením
  annotate("text", label = "Oblasť nezamietnutia H0", x = 0, y = -0.025, vjust=-1) +
  ggtitle("Znázornenie p-hodnoty")

4.1.4 Rozptyl

Príklad (Markechová 2011a, Príklad 6.10, str. 132)

Pri meraní koeficienta tepelnej vodivosti tehlovej steny boli namerané tieto hodnoty: 0,62; 0,64; 0,57; 0,61; 0,59; 0,57; 0,62; 0,59. Na základe nameraných výsledkov overte, či rozptyl tepelnej vodivosti tehlovej steny sa rovná predpísanej hodnote \(\sigma^2\) = 0,0004.
Riešenie:
Predpísanou hodnotou sa myslí to, aby nebola väčšia.

alpha <- 0.05
x <- c(0.62, 0.64, 0.57, 0.61, 0.59, 0.57, 0.62, 0.59)
n <- length(x)
TS <- (n-1) * var(x) / 0.0004
c(var_hat = var(x),
  test_statistika = TS,
  krit_hodnota = qchisq(1-alpha, df = n-1),
  p_hodnota = 1 - pchisq(TS, df = n-1)
  ) %>% round(5) # p-hodnota < 0.05
##         var_hat test_statistika    krit_hodnota       p_hodnota 
##         0.00064        11.21875        14.06714         0.12936
x <- seq(0, 20, length.out = 100)
dens <- dchisq(x, df = n-1)
ind <- list(r = which(x>=TS))
plot(x, dens, type = "l", main = "Znázornenie p-hodnoty")
polygon(x = c(x[ind$r[1]], x[ind$r], tail(x,1)), y = c(0, dens[ind$r], 0), 
        col="lightblue", border = NA)

alternatívne:

EnvStats::varTest(x, alternative = "greater", sigma.squared = 0.0004)

4.1.5 Príklady na precvičenie

Príklad (Markechová 2011a, Príklad 6.7, str. 128)

Podľa údajov na obale čokolády by mala byť jej čistá váha 125 gramov. Výrobca dostal niekoľko sťažností od zákazníkov, ktorí tvrdili, že váha čokolády je nižšia ako váha udávaná výrobcom. Pracovník oddelenia kontroly preto náhodne vybral 50 výrobkov a zistil, že ich priemerná váha je 122 gramov. Je známe, že smerodajná odchýlka sigma = 8,6 gramov. Overte na hladine významnosti alfa = 0,01, či sú sťažnosti zákazníkov opodstatnené.

Príklad (Markechová 2011a, Príklad 6.12, str. 135)

Úrad pre ochranu životného prostredia vydal nariadenie, ktorým určil priemyselným podnikom maximálne prípustné koncentrácie emisií rôznych látok. Horná hranica obsahu vinylchloridu vo vzduchu bola stanovená na 50 mg v kubickom km vo vzdialenosti do dvoch kilometrov od podniku, ktorý túto látku emituje. Pracovníci úradu vykonali kontrolu dodržiavania tohto nariadenia. Vo vzdialenosti dvoch kilometrov od podniku náhodne uskutočnili 100 meraní v rôznych dňoch a rôznych časoch. Priemerná koncentrácia vinylchloridu zistená pri týchto meraniach bola 54 mg a smerodajná odchýlka obsahu emitovaných látok bola 20 mg. Treba zistiť, či podnik dodržiava nariadenie úradu.

Príklad (Markechová 2011a, Príklad 6.9, str. 131)

Presnosť nastavenia automatického obrábacieho stroja vyrábajúceho určité výrobky je charakterizovaná rozptylom dĺžky súčiastok. Ak je táto hodnota väčšia ako 400 µm\(^2\) , automat treba znovu nastaviť. Výberová disperzia dĺžky 15 náhodne vybratých súčiastok z produkcie automatu je \(s^2\) = 680 µm\(^2\) . Treba posúdiť, či treba urobiť nové nastavenie stroja.

4.2 Dvojvýberové parametrické testy o parametroch normálneho rozdelenia

4.2.1 Nezávislé pozorovania

Príklad (Markechová 2011a, Príklad 6.14, str. 141)

Treba overiť efektívnosť nového spôsobu výučby vybraných tematických celkov matematiky. Preto výučba matematiky bola realizovaná v jednej triede experimentálnym spôsobom. Z tejto triedy boli náhodne vybraní žiaci, ktorí budú predstavovať experimentálnu skupinu. Žiaci z experimentálnej skupiny dosiahli v záverečnom teste, ktorý písali po realizácii experimentu, nasledujúce počty bodov: 22, 34, 26, 31, 26, 35, 25, 38, 36, 22, 23, 32. V ostatných triedach prebiehala výučba matematiky štandardným spôsobom. Z týchto tried boli náhodne vybraní žiaci, ktorí tvoria kontrolnú skupinu. Dosiahli v záverečnom teste nasledujúce počty bodov: 24, 33, 23, 20, 26, 32, 35, 21, 25.

x <- c(22, 34, 26, 31, 26, 35, 25, 38, 36, 22, 23, 32)
y <- c(24, 33, 23, 20, 26, 32, 35, 21, 25)

F-test rovnosti rozptylov

sx2 <- var(x)
sy2 <- var(y)
TS <- sx2/sy2
c(F_test_statistika = TS,
  krit_hodnota = qf(0.975, df1 = 11, df2 = 8),
  p_hodnota = pf(1/TS, df1 = 11, df2 = 8) + 1 - pf(TS, df1 = 11, df2 = 8)
)
## F_test_statistika      krit_hodnota         p_hodnota 
##         1.1346676         4.2434128         0.8516227

t-test stredných hodnôt

( sp2 <- (11*sx2+8*sy2)/(12+9-2) )  # pooled variance
## [1] 32.09942
TS <- (mean(x)-mean(y)) / sqrt(sp2*(1/12+1/9))  # testovacia štatistika
c(T_test_statistika = TS,
  krit_hodnota = qt(0.95, df = 12+9-2),  # jednostranný test
  p_hodnota = 1 - pt(TS, df=12+9-2)
)
## T_test_statistika      krit_hodnota         p_hodnota 
##         1.0451510         1.7291328         0.1545343
# vstavanými funkciami
var.test(x,y)  
## 
##  F test to compare two variances
## 
## data:  x and y
## F = 1.1347, num df = 11, denom df = 8, p-value = 0.8796
## alternative hypothesis: true ratio of variances is not equal to 1
## 95 percent confidence interval:
##  0.267395 4.157217
## sample estimates:
## ratio of variances 
##           1.134668

Nezamietame H0 o rovnosti rozptylov

##  29.16667  26.55556

4.2.2 Príklady na precvičenie

Príklad (Litschmanova online, Príklad 7.1)

Předpokládejme, že obsah nikotinu v cigaretách má normální rozdělení. Tabáková firma TAB prohlašuje, že jejich cigarety mají nižší obsah nikotinu než cigarety NIK. Pro ověření tohoto prohlášení bylo náhodně vybráno z produkce TAB 20 krabiček cigaret (po 20 kusech) a v nich bylo zjištěno průměrně 42,6 mg nikotinu (v jedné cigaretě). Výběrová směrodatná odchylka obsahu nikotinu v testovaných cigaretách TAB byla 3,7 mg. Ve 25 krabičkách (po 20 kusech) cigaret NIK bylo zjištěno průměrně 48,9 mg nikotinu na cigaretu. Výběrová směrodatná odchylka obsahu nikotinu v testovaných cigaretách NIK byla 4,3 mg. Ověřte tvrzení firmy TAB čistým testem významnosti.
