Prklad 1 (absencia)

Zadanie: 1. V tabuke s poty chbajcich iakov v kole poas pracovnch dn. Predpokladme, e nie je dvod aby v niektorch doch chbalo viac det. Vyberte vhodn teoretick rozdelenie, ktorho zhodu s rozdelenm dt budete testova.

H0 : Nie je toto rozdelenie normlove?

de pondelok utorok streda tvrtok piatok
poet 125-k 88+k 85 94 108

Rieenie:

k <- 8
x <- c(125+k, 88+k, 85, 94, 108)

Pre vizualizaciu dat si vykreslime histogram

hist(x)

Vypocitame si mean, sd aby sme dostali informacie a nasich vstupoch

xmean <- mean(x)
xsd <- sd(x)
n <- length(x)
c(xmean = xmean, xsd = xsd, n = n, range = range(x)) %>% round(2)
##  xmean    xsd      n range1 range2 
## 103.20  18.57   5.00  85.00 133.00

Pouzijeme Anderson-Darling test, ktory testuje ci H0: X pochadza z N(xmean,xsd).

goftest::ad.test(x, null = pnorm, mean = mean(x), sd = sd(x), estimated = F)  
## 
##  Anderson-Darling test of goodness-of-fit
##  Null hypothesis: Normal distribution
##  with parameters mean = 103.2, sd = 18.566098136119
##  Parameters assumed to be fixed
## 
## data:  x
## An = 0.33407, p-value = 0.9077

Na zaklade Anderson-Darling testu mozeme zamietnut nasu H0. Kedze z testu nam vyslo ze rozdelenie je normalovej distribucie.

Prklad 2 (domci poriadok)

Zadanie: Bolo skman dodriavanie iestich pravidiel domceho poriadku njomnkmi v socilnych bytoch pridelench mestom. Jednoduch nhodn vber 200 bytov odhalil nasledujce skutonosti o pote priestupkov.

poet_priestupkov 0 1 2 3 4 5 6
poet_bytov 31-k 51 70 32 9+k 5 2

Testujte, e vzorka pochdza z rozdelenia, v ktorom poet priestupkov (zo iestich monch priestupkov, n=6, p=?) na 1 byt je binomicky rozdelen nhodn premenn. Porovnajte teoretick a empirick poetnosti graficky.

Rieenie:

k <- 8
pocet_priestupkov <- c(0,   1,  2,  3,  4,  5,  6)
x <- c(31-k,    51, 70, 32, 9+k,    5,  2)


chisq.test(x, p=rep(1/length(x), length(x)))
## 
##  Chi-squared test for given probabilities
## 
## data:  x
## X-squared = 128.02, df = 6, p-value < 2.2e-16

Na zaklade vysledkov testu mozeme zamietnut hypotezu 0. Ze rozdelenie priestupkov na byt je binomicky nahodna premenna. Empiricka pocestnost

xsort <- sort(x)
Fn <- rank(xsort, ties.method="max")/length(x)
Ft <- pnorm(xsort, mean=mean(x), sd=sd(x))
plot(xsort, Fn, main="Empirick a parametrick distribun funkcia ")
lines(xsort, Ft)

Histogram

pocetnost <- list(
  skutocna = his$counts,

Prklad 4 (innos obojkov)

Zadanie: Porovnali sme dku innosti antiparazitnch obojkov dvoch znaiek urench pre menie plemen psov. Merania innosti v doch s dan tabukou. Na hladine vznamnosti =0.05 testujte hypotzu, e vbery s z rovnakho rozdelenia. V prpade zamietnutia nulovej hypotzy rozhodnite, ktor znaku by ste preferovali, ak ceny obojkov oboch znaiek s rovnak. Porovnajte aj graficky.

obojok_A 94 86 90 92 91 75+k 86 87 97 91 92 97
obojok_B 84 97 89 94 81+k 83 90 96 100 82

Rieenie:

k <- 8
x <- c(94   ,86 ,90 ,92 ,91 ,75+k   ,86 ,87 ,97 ,91 ,92 ,97) %>% sort()
y <- c(84   ,97 ,89 ,94 ,81+k   ,83 ,90 ,96 ,100    ,82) %>% sort()
Fx <- ecdf(x)
Fy <- ecdf(y)
z <- seq(80, 100, 1)
plot(z, Fx(z), type = "s", lwd = 2, ylab = NA)
lines(z, Fy(z), type = "s", col = 2, lwd = 2)

ks.test(x, y, alternative = "less")
